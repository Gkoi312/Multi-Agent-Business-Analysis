# 尽调回归样例集（第二阶段）

## 使用说明

- 目标：验证尽调流程稳定性与输出结构一致性。
- 执行方式：在 Dashboard 依次输入下列样例，检查任务是否可完成。
- 每个样例至少检查：
  - 是否生成 DOCX/PDF
  - 报告是否包含 `Company Overview / Business Breakdown / Scale and Development / Risk Assessment / Final Recommendation`
  - 风险统计是否展示 High/Medium/Low
